## Local Preview (Web Pages)

### Local Development (VitePress)

```powershell
npm install
npm run dev
```

### Generate Static Files (for GitHub Pages)

```powershell
npm run build
```

Build output goes to the `docs/` directory in the repository root (you can select "/docs" directory as deployment source in GitHub Pages).

Preview the build locally:

```powershell
npm run preview
```
