## 关键概念速览

- DesktopMemo 以“备忘录（Markdown）+ 待办事项（Todo）”为核心。
- 数据默认存放在应用目录下的 `/.memodata`（详见 [数据与备份](04_数据与备份.md)）。
- v2.3.0 起逐步引入本地 SQLite 数据库以提升性能与一致性（仍会保留 Markdown 正文文件）。
