# Keyboard Shortcuts

> Note: Available shortcuts vary by page; if they differ from your version, the in-app behavior takes precedence.

## Global

- `Ctrl + N`: Quickly create a new memo

## Editor Page Only

- `Tab`: Insert indentation (4 spaces)
- `Shift + Tab`: Decrease indentation
- `Ctrl + S`: Save current memo
- `Ctrl + Tab` / `Ctrl + Shift + Tab`: Switch between memos
- `Ctrl + F` / `Ctrl + H`: Find / Replace
- `F3` / `Shift + F3`: Find next / Find previous
- `Ctrl + ]` / `Ctrl + [`: Increase / Decrease indentation
- `Ctrl + D`: Duplicate current line downward
