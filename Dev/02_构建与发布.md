# 构建与发布

## 调试构建

```powershell
dotnet build DesktopMemo.sln --configuration Debug
```

## 发布构建

> 仓库内提供了 `build_exe.bat` 用于一键构建可执行文件，构建产物默认输出到 `artifacts/v<版本号>/bin`。

### 环境变量

- `ArtifactsVersion`：自定义构建版本号（用于输出路径）

## 发布目录（publish）

仓库内可能包含 `publish/DesktopMemo.exe`，通常用于本地测试或分发验证；正式发布以 GitHub Releases 为准。
