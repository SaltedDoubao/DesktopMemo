# 快捷键

> 说明：不同页面可用的快捷键不同；如与你的版本不一致，以应用内行为为准。

## 全局

- `Ctrl + N`：快速新建备忘录

## 仅在编辑页面可用

- `Tab`：插入缩进（4 个空格）
- `Shift + Tab`：减少缩进
- `Ctrl + S`：保存当前备忘录
- `Ctrl + Tab` / `Ctrl + Shift + Tab`：切换备忘录
- `Ctrl + F` / `Ctrl + H`：查找 / 替换
- `F3` / `Shift + F3`：查找下一个 / 上一个
- `Ctrl + ]` / `Ctrl + [`：增加 / 减少缩进
- `Ctrl + D`：向下复制当前行
