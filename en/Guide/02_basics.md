# Getting Started

## Memos

### Create

- Create a new memo from the main interface (you can also use keyboard shortcuts, see [Keyboard Shortcuts](03_shortcuts.md))

### Edit

- Supports Markdown
- From v2.4.0: Prompts for unsaved edits; no longer "auto-save"

### List Preview

- List supports quick content preview
- Supports search (if search service is enabled)

## Todo Items

- Quick add, check to complete, delete
- Completed items show strikethrough, can be undone or cleared
- Click "Memo/Todo" text in the title bar to switch modes

## Window Capabilities

### Pin Modes

- Normal pin
- Desktop pin
- Always on top

### Transparency & Mouse-Through

- Adjustable transparency
- Can enable mouse-through (suitable for "stick on desktop without interfering with operations")

> Effects may vary slightly depending on Windows version/theme.
