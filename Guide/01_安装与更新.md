# 安装与更新

## 获取应用

优先从 GitHub Releases 下载已打包的版本：

- Releases：[https://github.com/SaltedDoubao/DesktopMemo/releases](https://github.com/SaltedDoubao/DesktopMemo/releases)

> 如果你是开发者，需要本地编译运行，请看 [开发者指南](../Dev/README.md)。

## 系统要求

- Windows 10 版本 1903 及以上（WPF 依赖）
- x86_64
- 发布包通常自带 .NET 运行时；仅在自行编译时需要安装 .NET SDK

## 解压与运行

1. 解压到任意目录（建议避免放到需要管理员权限的路径）
2. 双击 `DesktopMemo.exe` 运行

首次运行会在可执行文件同级生成 `/.memodata` 数据目录。

## 更新方式

### 方式 A：覆盖更新（推荐）

1. 退出 DesktopMemo
2. 备份 `/.memodata`（至少备份 `settings.json` 与数据库文件）
3. 用新版本文件覆盖旧版本
4. 重新启动

### 方式 B：保留多版本并行

如果你希望保留多个版本，请确保每个版本的可执行文件目录互不相同，因为数据默认与 `DesktopMemo.exe` 同目录存放。

## 版本升级与迁移提示

应用启动时可能会执行数据迁移（例如将旧格式索引迁移到 SQLite）。如果你发现“文件修改时间被更新”等现象，请先阅读 [常见问题](05_常见问题.md) 的相关条目。
